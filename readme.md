# **🎬 CINEVASION - Projet de Recommandation de Films** {#cinevasion---projet-de-recommandation-de-films}

**CINEVASION** est une application interactive de recommandation de
films développée avec **Streamlit**. Elle permet aux utilisateurs de :

- Rechercher des films par titre.

- Découvrir des films similaires en utilisant des techniques de
  > similarité textuelle.

- Explorer des recommandations basées sur les notes et les genres.

- Visualiser les détails de chaque film.

## **📋 Fonctionnalités** {#fonctionnalités}

1.  **Page d\'accueil (Home)** :

    - Recherche de films par titre.

    - Application de filtres (décennie, genres, pays d\'origine).

    - Affichage des résultats sous forme de cartes avec affiches et
      > informations.

2.  **Page de Recommandations (Recommendations Page)** :

    - Recommandations basées sur les films les mieux notés.

    - Navigation vers les détails des films.

3.  **Page de Similarité (Similarity Page)** :

    - Recherche de films similaires à un film donné.

    - Calcul basé sur la similarité de contenu textuel (TF-IDF et
      > similarité cosinus).

4.  **Page de Détails (Details Page)** :

    - Affichage des détails d\'un film sélectionné, y compris les
      > affiches, genres, résumé, et plus encore.

    - Lien pour découvrir des films similaires.
